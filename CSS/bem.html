Metodologia BEM

Consiste en evitar conflictos de css para conocer que elementos estamos seleccionando en base a la clase.
Ponemos el nombre del contenedor, dos guiones bajos (__) y el nombre del elemento.
Ejemplo: en este formulario tenemos 2 inputs: texto y password, la clase dentro de cada elemento tiene la siguiente forma: 'form__input'

En caso querramos diferenciar a un elemento del resto, colocamos 2 guiones medios (--) y un nombre indicativo.
Ejemplo: Los elementos del siguiente formulario tienen la siguiente clase: 'contact-form__input'. Mientras que el primer elemento tiene la siguiente clase: 'contact-form__input--active'.
