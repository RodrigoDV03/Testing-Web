Primer titulo editado con CSS

La estuctura de css es la siguiente:
selector{
propiedad : valor
}
Significa:

  • Selector: Elementos que queremos cambiar.
  • Propiedad: Lo que le vamos a cambiar a los elementos seleccionados.
  • Valor: Su nombre lo dice

Tipos de selectores

  • Universal (*): Cambia todo
  • De tipo: Seleccionan directamente por los nombres de los elementos.
  • Por clases (.nombredelaclase): Se agrega el atributo class y modifica todos los elementos que presenten el nombre de la clase.
  • Por id (#): Se diferencia de clases por el caracter identificador. Estos se utilizan para identificar elementos unicos.
  • Por atributo ([]): Se seleccionan los atributos que cumplan las condiciones en los elementos html que lo contengan.
  • Descendientes: Cuando se tienen hijos. Primero se selecciona al contenedor y se va especificando (de derecha a izquierda o de fuera hacia dentro).
